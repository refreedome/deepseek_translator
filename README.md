# DeepSeek 智能翻译官

一个基于 DeepSeek 大模型的命令行翻译工具，输入中文即可获得英文翻译和优化建议。

## 功能特点

- 调用 DeepSeek-chat 模型进行高质量翻译
- 自动给出词汇和语法优化建议
- 交互式命令行界面，支持多轮翻译
- 代码结构清晰，适合入门学习 API 调用

## 技术栈

- Python 3.8+
- OpenAI SDK (兼容 DeepSeek API)
- DeepSeek 大模型 API
